Since Dev Mode requires a paid plan ($12-20/month), here are **completely free** ways to extract code and design specs from Figma.

## Best Free Options

### 1. **Figma to Code** Plugin ‚≠ê (Recommended)
- **Cost:** 100% Free & Open Source
- **What it does:** Converts Figma designs to HTML/Tailwind or Flutter/SwiftUI
- **How to use:**
  1. Open your Figma file
  2. Go to Plugins ‚Üí Search "Figma to Code"
  3. Select your frame/component
  4. Click "Generate Code"
  5. Copy the code and share with me in Cursor

**Install:** In Figma, go to `Plugins` ‚Üí `Browse all plugins` ‚Üí Search "Figma to Code"

### 2. **Export Kit** Plugin
- **Cost:** Free
- **What it does:** Copy code layer by layer, export to websites/apps
- **How to use:**
  1. Select any layer in Figma
  2. Run "Export Kit" plugin
  3. Copy code for that layer
  4. Repeat for each component

**Install:** Search "Export Kit" in Figma plugins

### 3. **HtmlGenerator** Plugin
- **Cost:** Free
- **What it does:** Generates clean HTML + CSS from selections
- **How to use:**
  1. Select element in Figma
  2. Run "HtmlGenerator" plugin
  3. Copy the generated HTML/CSS

**Install:** Search "HtmlGenerator" in Figma plugins

### 4. **pxCode** Plugin
- **Cost:** Free
- **What it does:** Automated responsive code generation (HTML/CSS/JS)
- **How to use:**
  1. Select frame or component
  2. Run "pxCode" plugin
  3. Get production-ready code

**Install:** Search "pxCode" in Figma plugins

## Manual Free Method (No Plugins)

If you prefer not to use plugins:

### Step 1: Export Assets
1. Right-click on images/icons ‚Üí **Export** ‚Üí Choose PNG (2x) or SVG
2. Save all assets to a folder
3. Share the folder with me

### Step 2: Extract Design Specs
1. **Colors:** Select colored element ‚Üí Copy hex code from right panel
2. **Typography:** Select text ‚Üí Copy font name, size, weight from right panel
3. **Spacing:** Hold `Option` (Mac) or `Alt` (Windows) ‚Üí Hover to see distances

### Step 3: Share with Me
Create a simple text file or just tell me:
```
COLORS:
- Primary: #3b82f6
- Background: #ffffff
- Text: #1f2937

TYPOGRAPHY:
- Heading: Inter, 32px, Bold
- Body: Inter, 16px, Regular

SPACING:
- Section padding: 80px
- Card gap: 24px
```

Then share:
- Figma file link (make it viewable/public)
- OR screenshots of each section
- OR the design spec file above
